INSTITUTO AZUL EDUCACIONAL - SITE HTML ESTÁTICO

Arquivos principais:
- index.html
- assets/css/style.css
- assets/js/main.js
- assets/img/*

Como usar:
1. Envie todo o conteúdo desta pasta para a raiz do domínio institutoazul.com.br.
2. O site é 100% estático: não precisa de PHP, banco de dados ou painel.
3. Os botões "Portal do Aluno", "Loja de Cursos" e "Ver curso" abrem a plataforma externa curso.study.
4. Para alterar curso, preço ou link, edite os cards no arquivo index.html.
